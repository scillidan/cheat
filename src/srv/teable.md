# [Teable](https://github.com/teableio/teable)

## install

[^1]

```sh
mkdir teable
cd teable
vim docker-compose.yml
vim .env
# Copy from https://help.teable.io/deployment/docker-compose#docker-compose
sudo docker-compose pull
```

## usage

```sh
sudo docker compose --env-file .env up -d
```
