## install

```sh
git clone --depth=1 https://github.com/coqui-ai/TTS
uv venv --python 3.10
.venv\Scripts\activate.bat
uv pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
uv pip install -e .
uv pip install transformers==4.40.2
```
