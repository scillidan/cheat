# [Yazi](https://github.com/sxyazi/yazi)

## install

```sh
# Arch
sudo pacman -S yazi
# Windows 10
scoop install yazi
```

## Personal do

```sh
ya pkg add gosxrgxx/flexoki-dark
ya pkg add gosxrgxx/flexoki-light
vim ~/.config/yazi/theme.toml
```

```toml
[flavor]
dark = "flexoki-dark"
light = "flexoki-light"
```
