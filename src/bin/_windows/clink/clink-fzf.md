# [FZF integration for Clink](https://github.com/chrisant996/clink-fzf)

## install

```sh
git clone --depth=1 https://github.com/chrisant996/clink-fzf
cd clink-fzf
rem "%CMDER_ROOT%\vendor\fzf.lua"
mklink "%CMDER_ROOT%\vendor\fzf.lua" "%cd%\fzf.lua"
```

### Optional

Change key bindings on [fzf.lua](https://github.com/scillidan/dotfiles/blob/main/.config/_windows_cmder/vendor/fzf.lua#L701).
