# [Ollama](https://ollama.com/)

## install

```sh
# Windows 10
scoop install ollama-full
```

## usage

```sh
ollama pull llama3.1:8b
ollama pull deepseek-r1:8b
# ollama pull qwen2.5-coder:7b
ollama pull qwen3:14b
ollama pull nomic-embed-text
ollama list
```

## config

[^1]

Add into environment:

```
OLLAMA_HOST=0.0.0.0
OLLAMA_ORIGINS=*
```
