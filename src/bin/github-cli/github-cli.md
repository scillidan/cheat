# [GitHub CLI](https://cli.github.com)

## install

```sh
# Arch
sudo pacman -S github-cli
# Windows 10
scoop install gh
```

## config

```sh
# Windows 10
set "GH_CONFIG_DIR=%USERPROFILE%\.config\gh"
```

## usage

```sh
gh auth login
```

```sh
gh extension install <user/repo>
gh extension remove <user/repo>
```
