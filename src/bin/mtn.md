# [mtn](https://gitlab.com/movie_thumbnailer/mtn)

## install

```sh
# Arch
yay -S --noconfirm mtn
# Windows 10
scoop install mtn
```

## usage

```sh
mtn -c 4 -r 4 -g 3 -k 000000 -w 1920 -i -t -D 4 -P -o "_mtn.png" "$1"
```
