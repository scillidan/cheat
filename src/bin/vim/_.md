# [Vim](https://github.com/vim/vim)

## install

```sh
# Arch
sudo pacman -S vim
# Windows 10
scoop install vim
```

## config

[^1]

```sh
vim ~/.vimrc
```

```
set tabstop=4
set shiftwidth=4
set noexpandtab
```
