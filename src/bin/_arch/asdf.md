# [asdf](https://github.com/asdf-vm/asdf)

## install

```sh
yay -S --noconfirm asdf-vm
```

## usage

```sh
asdf plugin add ruby https://github.com/asdf-vm/asdf-ruby
asdf install ruby 2.7.2
asdf set -u ruby 2.7.2
```
