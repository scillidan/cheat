# [NetworkManager](https://networkmanager.dev)

## install

```sh
sudo pacman -S networkmanager
```

## usage

```sh
# sudo systemctl stop netctl
# sudo systemctl disable netctl
sudo systemctl enable --now networkmanager
```

## optional

```sh
# Enable system tray
sudo pacman -S network-manager-applet
```
