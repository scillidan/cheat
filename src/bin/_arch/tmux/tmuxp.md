# [tmuxp](https://github.com/tmux-python/tmuxp)

## install

```sh
# Pip
uv tool install tmuxp
```

## usage

```sh
mkdir ~/.config/tmuxp
cd ~/.config/tmuxp
vim user.yaml
# Refer to https://tmuxp.git-pull.com/configuration/index.html
```

```sh
tmuxp load user
```
