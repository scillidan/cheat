# [tmux](https://github.com/tmux/tmux)

## install

```sh
sudo pacman -S tmux
```

## usage

### command

```sh
# Save session
tmux detach

# Restore session
tmux attach
```

```sh

chmod +x ~/.tmuxifier/bin/tmuxifier
~/.tmux/plugins/tpm/bin/install_plugins
```
