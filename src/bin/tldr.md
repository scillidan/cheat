# [tldr](https://github.com/tldr-pages/tldr)

## install

```sh
# Arch
sudo pacman tldr
# Windows 10
scoop install tldr
```

## usage

```sh
# Clear local repository then clone
tldr -c
# Pulls the latest commits from github.com/tldr-pages/tldr
tldr -u
```
