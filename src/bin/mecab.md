# [mecab](https://taku910.github.io/mecab/)

## install

1. Get `mecab-0.98.exe` from https://sourceforge.net/projects/mecab/files/mecab-win32/0.98/.
2. Run `mecab-0.98.exe`
	- Dictionary Charset → SHIFT-JIS
