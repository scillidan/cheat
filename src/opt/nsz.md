# [NSZ](https://github.com/nicoboss/nsz)

## install

```sh
# Arch
yay -S --noconfirm nsz
```

```sh
# From source
git clone --depth=1 https://github.com/nicoboss/nsz
cd nsz
uv venv
.venv\Scripts\activate
uv pip install -r requirements-gui.txt 
python nsz.py
```
