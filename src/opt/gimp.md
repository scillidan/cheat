# [GIMP](https://gimp.org)

## install

```sh
# Arch
sudo pacman -S gimp
# Windows 10
scoop install gimp
```

## usage

- 复制图层 → 选中新图层 → 颜色 → 去色 → 去色 → 模式 → luma
- 添加图层模板 → 选区工具 → 填充黑色
- 模式 → 点光/叠加/色相
