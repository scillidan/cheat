# [PureRef](https://pureref.com)

## install

```sh
# Arch
yay -S --noconfirm pureref
# Windows 10
scoop install pureref
```

## usage

- `Ctrl+Right` → `Ctrl+Down` → `Ctrl+f` → Resize by mouse wheel → `Ctrl+Shift+r`
