# [AnkiDroid](https://github.com/ankidroid/Anki-Android)

## config

- Settings → Sync → AnkiWeb
	- Custom sync server → `http://<your_host>:<port>`
	- AnkiWeb account → Used `<username>`, `<password>` set in Anki Sync Server's environment `SYNC_USER`
