# [Wez's Terminal](https://github.com/wezterm/wezterm)

## install

```sh
# Arch
sudo pacman -S wezterm
# Windows 10
scoop install wezterm
```

## setting

```sh
# Arch
~/.config/wezterm/wezterm.lua
# Windows 10
%USERPROFILE%\scoop\apps\wezterm\current\wezterm.lua
```
