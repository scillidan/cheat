# [Thorium](https://github.com/edrlab/thorium-reader)

## install

```sh
# Arch
yay -S --noconfirm thorium-reader-bin
# Windows 10
scoop install thorium-reader
```

## config

- Thorium → Catalogs → Add an OPDS feed:
	```
	Name: <name>
	Link: http://<your_host>/opds
	```
