# [Ditto](https://ditto-cp.sourceforge.io)

## install

```sh
scoop install ditto
```

## config

- Ditto
	- General
		- Maximum Number of Saved Copies → `100000`
		- Display Font
			```
			字体: 更纱终端书呆黑体-简
			大小: 小五
			```
		- Advanced → Text lines per clip → `1`
	- Keyboard Shortcuts → Activate Ditto → `Win-S-d`
