## optional

- Tools → Preferences → Syncing.
	- Self-hosted sync server `http://<your_host>:<port>`.
		- If `<your_host>` don't work, try used `ip`.
	- AnkiWeb Account → Login → Used `<username>`, `<password>` set in Anki Sync Server's environment `SYNC_USER`.
