# [ES-DE](https://gitlab.com/es-de/emulationstation-de)

## install

```sh
# Arch
yay -S --noconfirm emulationstation-de
```

### Windows 10

1. Get `ES-DE_*-x64_Portable.zip` from [Releases](https://gitlab.com/es-de/emulationstation-de/-/releases).
2. Decompress it to `ES-DE/`.
