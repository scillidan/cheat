# [Material for MkDocs](https://github.com/squidfunk/mkdocs-material)

## install

```sh
mkdir <site>
cd <site>
uv venv
.venv\Scripts\activate.bat
uv pip install mkdocs-material
```

## usage

```sh
mkdocs new .
subl config.yaml
```

```yaml
theme:
  name: material
```
