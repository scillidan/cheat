# [mdBook](https://rust-lang.github.io/mdBook)

## install

```sh
cargo install --force mdbook
```

## usage

```sh
mdbook init <yourbook>
cd <yourbook>
# Serve
mdbook serve --open
# Build
mdbook build
```
