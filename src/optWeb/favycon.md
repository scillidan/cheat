## install

```sh
git clone --depth=1 https://github.com/ruisaraiva19/favycon
cd favycon
subl .eslintrc.json
```

```json
	"rules": {
		"prettier/prettier": ["error", {'endOfLine': 'auto'} ]
```

```sh
yarn install
yarn build
```

## usage

```sh
yarn start -- -p 4321
```
