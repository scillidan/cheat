## install

```sh
git clone --depth=1 https://github.com/bbodi/notecalc3
cd notecalc3
# rustup override set nightly-2020-11-17
# cargo install --force wasm-pack
# wasm-pack build --release --target no-modules frontend-web
```

```sh
serve -s . -p 4321
```
