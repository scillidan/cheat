## install

```sh
git clone --depth=1 https://github.com/tone-row/flowchart-fun
cd flowchart-fun
pnpm install
pnpm build
```

## usage

```sh
serve -s app/build -p 4321
```

```sh
# PM2
pm2 serve app\build\ 4321 --name flowchart-fun --spa
```
