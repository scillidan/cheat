## install

```sh
git clone --depth=1 https://github.com/siddharthroy12/recoded
cd recoded
npm install --legacy-peer-deps
npm run build
```

## usage

```sh
serve -s build -p 4321
# pm2 serve build 4321 --name recoded --spa
```
