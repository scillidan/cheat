# [SVGOMG](https://github.com/jakearchibald/svgomg)

## install

```sh
git clone --depth=1 https://github.com/jakearchibald/svgomg
cd svgomg
npm install
npm run build
```

## usage

```sh
serve -s build -p 4321
```
