# [KeyMap](https://github.com/Gustice/AHK-KeyMap)

## install

```sh
git clone --depth=1 https://github.com/Gustice/AHK-KeyMap
cd AHK-KeyMap
<path_to>\autohotkey1.1\AutoHotkeyU64.exe KeyMap.ahk
```

## config

```sh
subl KeyMap.ahk
```
