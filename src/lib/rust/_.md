# [Rust](https://www.rust-lang.org)

## install

[^1]

```sh
# Arch
sudo pacman -S rustup
# Windows 10
scoop install rustup
# Ubuntu 22 ARM
sudo apt install rustc cargo
```

## usage

```sh
rustup default stable
rustup update
```

[^1]: [How to Install Rust on Ubuntu](https://phoenixnap.com/kb/install-rust-ubuntu)
